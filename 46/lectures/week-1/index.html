Course expectations

This is your course. So you decide how much you want to get out of it and how much you're willing to put in. This is what success looks like:

  • Missing no more than two classes
  • Doing all your homework
  • Taking written notes during lectures
  • Helping other students with topics you understand
  • Presenting a final project on the last day

Notice that success is not defined by the quality of your work. You're not here to learn everything about how to build web applications. Some people learn material faster than others. Some people have more prior knowledge than others. So define your own expectations of success.

Hello World

  1. Confirm that you have not created any repos on GitHub thus far
  2. Create a new folder "fewd-46"
  3. Open that folder in Atom
  4. Create a file in that folder "index.html"
  5. Add the folder to GitHub, which will make it a repository
  6. Make your first commit!
  7. Publish to GitHub
  8. View your repo on GitHub
  9. "Cut" a branch called "gh-pages" and publish it to GitHub
  10. See your page at http://{ your user name }.github.io/{ your repository name }/ for example: http://avand.github.io/fewd/

Creating your profile

Homework

  1. "Fork" the course curriculum repo: github.com/avand/fewd
  2. Clone the repo in GitHub Desktop
  3. Open the repo in Atom
  4. Find the file 46/index.html and find the part that references you!
  5. Create the necessary links (including the link to your profile)
  6. Commit and sync your changes
  7. Submit a pull request from your repo back to avand/fewd
